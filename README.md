# nCoV PH Tracker Scraper

#### Prerequisites

- Python
- VS Code (for coding convinient)
- Added SSH keys
- Chromedriver has been downloaded.
- Chromedriver path added to $PATH

### Getting Started

```
git clone git@github.com:bjmbn/ncov-ph-tracker-scraper-python.git
```

